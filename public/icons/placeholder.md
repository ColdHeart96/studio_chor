# Icônes PWA

Placez ici vos icônes PNG :
- `icon-192.png` — 192×192 px (icône Android / PWA)
- `icon-512.png` — 512×512 px (icône splash screen)
- `apple-touch-icon.png` — 180×180 px (iOS)

## Génération rapide

Vous pouvez créer ces icônes à partir d'un SVG ou d'un PNG source avec :
- https://pwa-asset-generator.netlify.app/
- https://realfavicongenerator.net/
- La commande : `npx pwa-asset-generator logo.png public/icons`

Le logo recommandé est un disque doré avec le symbole ♪ sur fond sombre (#080812).
